## Project Overview

The 'Bus Booking System' is a web-based application designed to automate the process of booking bus tickets.
  
It provides separate access for 'Users' and 'Administrators', ensuring secure management of bookings, buses, and schedules.  
The system uses a relational database (SQL) to store user details, bookings, buses, and seat information.

## Objectives
- To automate the bus ticket booking process
- To provide secure user and admin authentication
- To manage bus schedules and seat availability efficiently
- To reduce errors and delays associated with manual booking systems

## System Features

### User Features
- User registration and login
- View available buses and routes
- Select travel date and bus
- Book seats
- View booking details

### Admin Features
- Admin login authentication
- Add, update, and delete bus details
- Manage routes and schedules
- View all bookings
- Manage system data stored in the            database
